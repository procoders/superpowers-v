## Design constraints (non-negotiable MUST-HANDLE)

1. **`tracked_files()` (`scripts/compound-v-memory.py:238`) must be EXTENDED, not replaced.** Add a scoped second `git ls-files -z -- AGENTS.md CLAUDE.md CONVENTIONS.md DESIGN.md` union alongside the `DOCS_REL` listing. Do NOT widen `DOCS_REL` (would index the whole repo). Preserve the fail-closed `[]` on git error (`:258`) and the worktree-only-trust-git rule (`:247`).
2. **`doc_type_for()` (`:133`) must gain explicit cases for the four root files.** Today any non-`docs/superpowers/` path returns `parts[0]` ÔÇö i.e. the filename itself (`AGENTS.md` Ôćĺ `"AGENTS.md"`, verified). Map to clean labels (`agents`/`claude`/`conventions`/`design`). Existing selftest assertions (`:964-966`) must still pass.
3. **The staleness manifest hashes CITED SOURCE files** ÔÇö a different population than `indexed_files.content_hash` (which hashes the generated doc). REUSE `file_sha()` (`:268`, streaming sha256, `""` on error); add NEW state, do not overload `indexed_files`.
4. **`.onboard-manifest.json` is `.json` and is NOT indexed** (the filter is `.md`/`.jsonl` only, `:255`). Correct and desirable ÔÇö it is machinery, not recall prose. Do NOT name it `.jsonl` (would be chunked per-line as recall noise via `chunk_jsonl`, `:193`). Keep `.json`, explicitly out of the index.
5. **WRITE Ôćĺ commit Ôćĺ INDEX ordering must be explicit.** Recall and the scope gate require git-tracked files (engine `:247-258`). Generated docs must be `git add`+committed BEFORE the auto `/v:memory-refresh`, or the index stays empty for them. (Spec gap closed in rev.2.)
6. **The SessionStart staleness line MUST be a conditional append to `$banner` BEFORE the jq platform switch (`hooks/session-banner.sh:25`)** ÔÇö never a second jq emit. It MUST fail silent: `set -euo pipefail` (`:13`) aborts the whole banner on any non-zero exit, so guard the python probe like `hooks/memory-refresh.sh` does (`command -v python3 >/dev/null 2>&1 || exit 0`, `|| true`). It writes nothing.
7. **Blocking secret scan REUSES `SECRET_RE`/`PEM_RE` (`:53-63`)** as the detector but implements REFUSAL (new behavior). `redact()` (`:128`) MASKS and returns text ÔÇö it never blocks. One detector, two policies; do not fork a second secret list.
8. **`flock` discipline:** if `/v:onboard --refresh` calls indexing, it must go through `cmd_refresh`'s existing lock (`:565`), not a second sqlite writer.
9. **Command wiring:** new `commands/v-onboard.md` (frontmatter `description:` required; `{{args}}` parses `--refresh`); NO `.claude-plugin/plugin.json` registration (commands auto-discovered). The `/v:init` suggestion goes in `commands/v-init.md` Step 5 (Report) as report text, never a chain. New command file must pass `.cclintrc.json` frontmatter lint.
10. **Engine = Task 0 / serial.** `scripts/compound-v-memory.py` is the shared foundation the command, hook, and manifest all consume; it must land before the dependents.
