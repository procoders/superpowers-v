# MCP recommender (v2.5.1) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:test-driven-development for the toolkit tasks. Steps use checkbox (`- [ ]`) tracking.

**Goal:** Add a deterministic MCP/external-tool recommender to `/v:onboard` — signal→tool with a CLI-over-MCP bias, least-privilege `.mcp.json` via diff+confirm, and a warn-only lethal-trifecta note.

**Architecture:** All logic lands in the existing `scripts/compound-v-onboard.py` as a new `recommend-mcp` subcommand backed by a **static curated `MCP_RULES` table** + a `recommend_mcp(repo)` detector + a `mcp_json_config()` additive writer. Prose wiring in `onboarding.md` + `v-onboard.md`. Stdlib-only, mirrors the existing `check()`/tempdir selftest pattern.

**Tech Stack:** Python 3.9 stdlib, Markdown skill/command docs, JSON (`.mcp.json`, version files).

## Global Constraints
- **CLI-over-MCP bias** — `github.com` remote ⇒ `gh` CLI, **never** a GitHub MCP server (broad-PAT toxic-flow).
- **Least-privilege pre-filled** — read-only / restricted / project-ref defaults on every MCP row.
- **`.mcp.json` only via diff + confirm** — never auto-apply; merge **additively**, never clobber unrelated servers.
- **Lethal-trifecta warn-only** — named warning + specific remedy; no hard refusal.
- **Deterministic + evidence-cited** — static curated table; each recommendation cites its signal; unknown stack ⇒ empty set (no invented tools).
- **Currency-verified (2026-07-01, WebSearch):** Supabase MCP `@supabase/mcp-server-supabase` (`--read-only --project-ref`); Playwright MCP `@playwright/mcp` pinned `>=0.0.40` (CVE-2025-9611); Postgres MCP `crystaldba/postgres-mcp` (`--access-mode=restricted`); Context7 `@upstash/context7-mcp`; Sentry `@sentry/mcp-server`.
- **Version lockstep:** `plugin.json` + `marketplace.json` → `2.5.1` (CI enforces).
- No fabricated metrics.

```python
FASTMOVING = ("react", "next", "vue", "svelte", "@sveltejs/kit", "nuxt", "tailwindcss",

def _git_remote(repo):
    try:
        out = subprocess.run(["git", "-C", repo, "remote", "-v"],
                             capture_output=True, text=True, timeout=15)
    except (OSError, subprocess.SubprocessError):
        return None
    return out.stdout.strip() if out.returncode == 0 else None


def _pkg_deps(repo):
    """All dependency names from package.json (deps + devDeps). ({} , evidence|None)."""
    pj = os.path.join(repo, "package.json")
    if not os.path.isfile(pj):
        return {}, None
    try:
        with open(pj) as fh:
            data = json.load(fh)
    except (OSError, ValueError):
        return {}, None
    deps = {}
    for key in ("dependencies", "devDependencies"):
        d = data.get(key)
        if isinstance(d, dict):
            deps.update(d)
    return deps, "package.json"


def _rec(rule_id, evidence):
    r = dict(MCP_RULES[rule_id])
    r["evidence"] = evidence
    return r


def recommend_mcp(repo):
    """Deterministic signal -> tool recommender. Returns {recommendations, warnings}. Each
    recommendation is an MCP_RULES row + `evidence`; unknown stack yields an empty set."""
    recs, seen = [], set()

    def add(rule_id, evidence):
        if rule_id not in seen:
            recs.append(_rec(rule_id, evidence))
            seen.add(rule_id)

    remote = _git_remote(repo)
    if remote and "github.com" in remote:
        add("github", "git remote references github.com")

    deps, dep_ev = _pkg_deps(repo)
    if any(d.startswith("@supabase/") for d in deps):
        add("supabase", dep_ev)
    if any(d.startswith("@sentry/") for d in deps):
        add("sentry", dep_ev)
    if any(d in deps for d in ("pg", "prisma", "@prisma/client")):
        add("postgres", dep_ev)
    if any(d in deps for d in FASTMOVING):
        add("context7", dep_ev)

    for name in ("playwright.config.ts", "playwright.config.js", "playwright.config.mjs"):
        if os.path.isfile(os.path.join(repo, name)):
            add("playwright", name)
            break

    warnings = [{"id": r["id"],
                 "message": "%s combines private-data access with write capability (lethal-trifecta risk)." % r["tool"],
                 "remedy": TRIFECTA_REMEDY}
                for r in recs if r["trifecta"]]
    return {"recommendations": recs, "warnings": warnings}
```

```python
def mcp_json_config(recommendations, existing=None):
    """Additive .mcp.json for the kind=='mcp' recommendations. Never clobbers an existing
    same-named server; CLI recs (e.g. gh) are excluded (surfaced as setup instructions)."""
    servers = dict((existing or {}).get("mcpServers", {}))
    for r in recommendations:
        if r.get("kind") != "mcp" or r["id"] in servers:
            continue
        servers[r["id"]] = {"command": "npx", "args": ["-y", r["package"]] + list(r["flags"])}
    return {"mcpServers": servers}
```
