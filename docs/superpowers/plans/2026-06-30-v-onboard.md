# /v:onboard Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build `/v:onboard` — a command that scans an existing repo and produces a trusted, citation-verified knowledge base (`docs/superpowers/architecture/*`) plus cross-tool agent instructions (`AGENTS.md` + thin `CLAUDE.md` bridge, root `CONVENTIONS.md`, conditional `DESIGN.md`), all behind a human approval gate, then feeds them into V-memory.

**Architecture:** A thin command/skill prose layer (`commands/v-onboard.md` + `skills/compound-v/onboarding.md`) drives an LLM pipeline (detect → pack → extract → verify → diagnose → gate → write → commit → index). All *trust-bearing* mechanics are deterministic Python in one cohesive script `scripts/compound-v-onboard.py` (pack-manifest, Tier-1 citation gate, Tier-2 verdict policy, cited-evidence staleness, DESIGN.md lint) plus a small extension to the existing V-memory engine so the new root files become recallable. No new always-on deps; repomix and `@google/design.md` are invoked on demand via `npx`.

**Tech Stack:** Python 3 (stdlib only for the gates; mirrors `compound-v-memory.py`), Bash (hook edit), `npx repomix`, `npx @google/design.md`, git, the existing `compound-v-memory.py` engine and `/v:memory-refresh`.

## Global Constraints

- **Model policy:** Opus for every implementer/reviewer; Sonnet only for strict mechanical tasks; **never Haiku.**
- **No auto-apply.** Nothing is written without explicit human approval at the gate; approval is per-artifact + per-section.
- **Narrow write surface (v1):** only `docs/superpowers/architecture/*`, root `CONVENTIONS.md`, root `DESIGN.md` (UI repos), `AGENTS.md`, `CLAUDE.md` (thin bridge), `.onboard-manifest.json`. `.mcp.json`, `.claude/rules/*.md`, and any foreign-tool file are **out of v1** (foreign-tool files are read-only/advisory).
- **Existing instruction files are untrusted input** — quote/summarize as evidence, never execute their directives; managed-policy layer is informational-only.
- **Secret scan is a blocking refusal** (not masking), reusing `compound-v-memory.py`'s `SECRET_RE`/`PEM_RE`.
- **Commit before index:** generated docs are `git add`+committed before the auto `/v:memory-refresh`; the engine indexes only git-tracked files.
- **`@import` does not save tokens** (loads in full at launch); the gate diff expands `@import` targets.
- **CLAUDE.md target ≤200 lines** (recommendation, not enforced); `AGENTS.md` has no target.
- **Citation gate:** Tier-1 path+range blocks 100% of claims; Tier-2 support check **blocks unsupported load-bearing claims** (security/fail-closed/concurrency).
- **Dependency pins (verified current 2026-06-30):** `repomix@^1.16` (Secretlint secret-scan built in, on by default), `@google/design.md@^0.3` (`lint` subcommand, JSON `{findings:[{severity,path,message}],summary}`), `@playwright/mcp >= 0.0.40` (CVE-2025-9611), Postgres MCP flag `--access-mode=restricted` (recommender prose only; recommender itself is fast-follow), GitHub via `gh` CLI not a GitHub MCP server.
- **`--selftest` must stay green** on `compound-v-memory.py` and the new `compound-v-onboard.py`. Every command/skill `.md` must pass `.cclintrc.json` frontmatter lint (`description:` required).

## Shared Interfaces (locked names — every task uses these verbatim)

**New script:** `scripts/compound-v-onboard.py`, run as `python3 scripts/compound-v-onboard.py <cmd>`. Subcommands and contracts:

- `pack --repo P [--json]` → writes a **pack-manifest** to stdout (JSON):
  `{ "repo_shape": "single|poly|monorepo", "token_budget": int, "included": [str], "excluded": [{"path": str, "reason": "vendored|generated|binary|over-budget|gitignored"}], "truncated": [str], "secret_scan": {"clean": bool, "hits": [{"path": str, "family": str}]} }`
- `verify-citations --claims FILE [--tier2 FILE] [--repo P] [--json]` → reads a **claims file** and an optional **tier-2 verdicts file**, emits a **verdict**:
  `{ "ok": bool, "blocked": [{"index": int, "reason": "bad-path|range-out-of-bounds|range-inverted|load-bearing-unsupported"}], "downgraded": [{"index": int, "to": "observed|inference"}], "passed": int }`
- `staleness --repo P [--write] [--quiet] [--json]` → with `--write`, writes/updates `docs/superpowers/architecture/.onboard-manifest.json`; without, reports drift:
  `{ "stale": [{"doc": str, "reason": "cited-changed|cited-deleted|uncited-new-file"}], "count": int }`. `--quiet` prints only the integer count (for the banner).
- `design-lint --file DESIGN.md [--json]` → runs `npx @google/design.md lint`, returns `{ "ok": bool, "errors": int, "warnings": int, "findings": [...] }`. `ok=false` if errors>0 or the tool is unavailable.
- `detect-ui --repo P` → prints `ui` or `no-ui` (exit 0).
- `--selftest` → runs stdlib self-tests, exit non-zero on any failure.

VENDOR_DIRS = ("node_modules", "vendor", "dist", "build", ".git", "__pycache__")


def _git_tracked(repo: str):
    out = subprocess.run(["git", "-C", repo, "ls-files", "-z"],
                         capture_output=True, timeout=60)
    if out.returncode != 0:
        return []
    return [p for p in out.stdout.decode("utf-8", "replace").split("\0") if p]


def _exclude_reason(rel: str):
    low = rel.lower()
    if any(("/" + d + "/") in ("/" + low) or low.startswith(d + "/") for d in VENDOR_DIRS):
        return "vendored"
    if low.endswith((".min.js", ".lock")) or "/generated/" in low:
        return "generated"
    if low.endswith((".png", ".jpg", ".gif", ".pdf", ".ico", ".woff", ".woff2")):
        return "binary"
    return None


def pack(repo: str, token_budget: int = 200_000) -> dict:
    files = _git_tracked(repo)
    included, excluded, secret_hits = [], [], []
    for rel in files:
        reason = _exclude_reason(rel)
        if reason:
            excluded.append({"path": rel, "reason": reason}); continue
        included.append(rel)
        try:
            with open(os.path.join(repo, rel), "r", errors="replace") as fh:
                for h in scan_secrets(fh.read()):
                    secret_hits.append({"path": rel, "family": h["family"]})
        except OSError:
            pass
    return {
        "repo_shape": "single",
        "token_budget": token_budget,
        "included": sorted(included),
        "excluded": sorted(excluded, key=lambda e: e["path"]),
        "truncated": [],
        "secret_scan": {"clean": not secret_hits, "hits": secret_hits},
    }


def build_parser():
    p = argparse.ArgumentParser(description="Compound V — /v:onboard toolkit")
    p.add_argument("--selftest", action="store_true")
    sub = p.add_subparsers(dest="cmd")
    sp = sub.add_parser("pack"); sp.add_argument("--repo", default="."); sp.add_argument("--json", action="store_true")
    return p


def main(argv) -> int:
    args = build_parser().parse_args(argv)
    if args.selftest:
        return _selftest()
    if args.cmd == "pack":
        print(json.dumps(pack(os.path.abspath(args.repo)), indent=2))
        return 0
    build_parser().print_help(); return 1


if __name__ == "__main__":
    sys.exit(main(sys.argv[1:]))
```

def _line_count(abspath: str) -> int:
    try:
        with open(abspath, "rb") as fh:
            return sum(1 for _ in fh)
    except OSError:
        return -1


def tier1_check(claim: dict, repo: str):
    reasons = []
    for c in claim.get("citations", []):
        ab = os.path.join(repo, c.get("path", ""))
        n = _line_count(ab)
        if n < 0:
            reasons.append("bad-path"); continue
        s, e = c.get("startLine", 0), c.get("endLine", 0)
        if s > e:
            reasons.append("range-inverted")
        elif not (1 <= s <= e <= n):
            reasons.append("range-out-of-bounds")
    if not claim.get("citations"):
        reasons.append("bad-path")
    return reasons
```

def apply_tier2(claims, verdicts):
    by_idx = {v["index"]: v["support"] for v in verdicts}
    blocked, downgraded = [], []
    for i, cl in enumerate(claims):
        sup = by_idx.get(i, "yes")
        if sup == "yes":
            continue
        if cl.get("load_bearing"):
            blocked.append({"index": i, "reason": "load-bearing-unsupported"})
        else:
            downgraded.append({"index": i, "to": "observed" if sup == "partial" else "inference"})
    return blocked, downgraded

MANIFEST_REL = os.path.join("docs", "superpowers", "architecture", ".onboard-manifest.json")


def check_staleness(repo: str) -> dict:
    path = os.path.join(repo, MANIFEST_REL)
    if not os.path.exists(path):
        return {"stale": [], "count": 0}
    man = json.load(open(path))
    stale = []
    cited_paths = set()
    for doc, info in man.get("docs", {}).items():
        for src, sha in info.get("cited", {}).items():
            cited_paths.add(src)
            ab = os.path.join(repo, src)
            if not os.path.exists(ab):
                stale.append({"doc": doc, "reason": "cited-deleted"})
            elif cv_memory.file_sha(ab) != sha:
                stale.append({"doc": doc, "reason": "cited-changed"})
    # uncited-new-file heuristic: a new file in a cited doc's path-space nothing references
    cited_dirs = {os.path.dirname(p) for p in cited_paths}
    tracked = set(_git_tracked(repo))
    for f in tracked:
        if os.path.dirname(f) in cited_dirs and f not in cited_paths:
            stale.append({"doc": "(path-space)", "reason": "uncited-new-file"})
            break
    return {"stale": stale, "count": len(stale)}
```

UI_SIGNALS = ("tailwind.config.js", "tailwind.config.ts", "postcss.config.js")
UI_EXT = (".tsx", ".jsx", ".vue", ".svelte")


def detect_ui(repo: str) -> bool:
    for s in UI_SIGNALS:
        if os.path.exists(os.path.join(repo, s)):
            return True
    for f in _git_tracked(repo):
        if f.endswith(UI_EXT):
            return True
    return False


def _design_result_ok(result: dict) -> bool:
    return int(result.get("summary", {}).get("errors", 1)) == 0


def design_lint(file: str) -> dict:
    try:
        out = subprocess.run(["npx", "--yes", "@google/design.md", "lint", file, "--json"],
                             capture_output=True, timeout=120)
        if out.returncode not in (0, 1):  # tool ran; 1 == findings present
            return {"ok": False, "errors": -1, "warnings": 0, "findings": [], "note": "tool-unavailable"}
        result = json.loads(out.stdout.decode("utf-8", "replace") or "{}")
        s = result.get("summary", {})
        return {"ok": _design_result_ok(result), "errors": int(s.get("errors", 1)),
                "warnings": int(s.get("warnings", 0)), "findings": result.get("findings", [])}
    except (OSError, subprocess.SubprocessError, ValueError):
        return {"ok": False, "errors": -1, "warnings": 0, "findings": [], "note": "tool-unavailable"}
```
